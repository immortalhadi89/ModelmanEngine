# ModelmanEngine
🚀 **ModelmanEngine** helps Iranians search the internet freely, even with restrictions and after the 2026 war – extracting full text, images/GIFs, and all links from **most webpages** (excluding those protected by aggressive bot defenses).
ℹ️ **To get started:** First, either **fork** this repository (for public use) or click the **plus icon** and choose **Import repository** (for private use).
> 🛠️ **How to use:**  
> - 💻 **Windows & Android (desktop mode on):** Go to **Actions** → **All workflows ▽** → **🔍 Search** → **Run workflow ▽** → pick settings → click green **Run workflow**.  
> - 📱 **Android (desktop mode off):** Tap **More ▽** → **Actions** → **All workflows ▽** → **🔍 Search** → **Run workflow ▽** → pick settings → tap green **Run workflow**.  
> - ♻️ **Factory reset:** Run **♻️ Reset Factory** workflow → deletes everything → back to day one.
